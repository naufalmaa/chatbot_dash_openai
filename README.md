# chatbot_dash_openai

Simple Chatbot

This is dummy model of simple chatbot with help of OpenAI API and Dash App.
This app can help you ask anything you want, like ChatGPT.

This app needs libraries:
- openai (v 0.27.0)
- dash
- dash mantine components
- dash iconify
